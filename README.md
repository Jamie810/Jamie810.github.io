# 个人主页

一个现代化、响应式的个人主页模板，展示你的技能、项目和联系方式。

## 功能特点

- ✨ 现代化的设计风格
- 📱 完全响应式，支持移动端和桌面端
- 🎨 流畅的动画效果
- 🚀 快速加载，性能优化
- 📝 易于自定义和修改

## 文件结构

```
个人主页/
├── index.html      # 主HTML文件
├── styles.css      # 样式文件
├── script.js       # JavaScript交互文件
└── README.md       # 说明文档
```

## 使用方法

1. **直接打开**
   - 双击 `index.html` 文件即可在浏览器中查看

2. **使用本地服务器（推荐）**
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js (需要安装http-server)
   npx http-server
   ```
   然后在浏览器中访问 `http://localhost:8000`

## 自定义内容

### 1. 修改个人信息

在 `index.html` 中修改以下内容：

- **名字**: 搜索 `你的名字` 并替换
- **职业描述**: 修改 `hero-subtitle` 中的文本
- **关于我**: 在 `#about` 部分修改介绍文字
- **统计数据**: 修改 `about-stats` 中的数字和标签

### 2. 更新技能

在 `#skills` 部分，修改技能卡片：
- 技能名称
- 技能描述
- 技能水平（修改 `skill-bar` 的 `width` 属性）

### 3. 添加项目

在 `#projects` 部分，复制项目卡片并修改：
- 项目名称
- 项目描述
- 技术标签
- GitHub和演示链接

### 4. 更新联系方式

在 `#contact` 部分：
- 修改邮箱地址
- 更新社交媒体链接（GitHub, LinkedIn, Twitter等）
- 确保所有链接都是有效的

### 5. 更换头像

你可以：
- 替换 `<i class="fas fa-user"></i>` 为你的头像图片
- 或者使用在线头像服务

### 6. 修改颜色主题

在 `styles.css` 的 `:root` 部分修改CSS变量：

```css
:root {
    --primary-color: #6366f1;    /* 主色调 */
    --secondary-color: #8b5cf6;  /* 次要色调 */
    /* ... 其他颜色变量 */
}
```
